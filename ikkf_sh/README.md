# IKKF_SH — Show Me

> Продолжение сцены из Матрицы: Нео сказал "I know kung-fu" (IKKF), Морфиус ответил "Show me" (SH).

**IKKF_SH** — это слой действий и навыков поверх памяти IKKF. Если IKKF — это *знание*, то IKKF_SH — это *умение применять знание*.

---

## Модули

| Модуль | Назначение |
|--------|-----------|
| `reasoning/` | Chain-of-thought, verification, self-critique |
| `skills/` | Создание, хранение, эволюция навыков |
| `planner/` | Декомпозиция задач, выбор инструментов |
| `memory/` | Working Memory — краткосрочный контекст |
| `search/` | Deep Search — ежедневный сбор данных |
| `verification/` | Fact-checking, confidence scoring, rollback |
| `agents/` | Self-Replication, Multi-Agent Orchestration |
| `integrations/` | Telegram, Device Control, External APIs |

## Принципы

1. **Действие > Знание** — не просто хранить, а применять
2. **Верификация** — каждый шаг проверяется
3. **Саморазвитие** — навыки эволюционируют через практику
4. **Контроль** — пользователь всегда главный
